# Division template historical names system. Is a new method of naming the divisions based on the names-group assigned to it's template.
# If we run out of all historical names, the names will be assigned in the old way.
#
# Each group has a made up tag. Use it if you want to append more, or replace existing names with the extra txt files (modding).
#
# for_countries - lists all countries that can use it. If empty, or entire tag is missing, all countries in the world can use it.
#
# can_use - is a trigger that locks/unlocks the group under specific circumstances. The trigger is in a country scope.
#
# division_types - is a list of tokens to corresponding unit types. A player can in fact use any group of names for a div.template
#			however this tag is a helper for an automated choice (for AI, or if the group must switch on it's own, because
#			for example the current one is no longer available due to the can_use trigger saying so).
#			In automated choice, the division template must have at least 1 of the following types for it to be chosen.
#
# fallback_name - Is going to be used if we run out of the scripted historical names. If you want to use the old division naming
#			mechanics to be used for fallbacks, then just skip this option.
#
# unordered - It's a list of historical division names that did not have a number. Regardless if such names happened in history or not
#			this option is available here mainly for a mods.
#
# ordered - Is a list of all historical names.
#			Numbers must start from 1 and up.
#			Numbers can't repeat in one scope.
#			If you script the name for this group, at the same number (for example in a mod in another file), the name will be override.
#			All arguments between the brackets must be separated by spaces. Each argument is wrapped in "quotas".
#			1st argument = The name. It must contain either:
#						%d (for decimal number placement)
#						%s (for string number placement - ROMAN numbers like XIV).
#			2nd argument is optional = A localizable text describing this historical division. The text will be visible in the tooltip
#									where you choose the historical division name.
#			3rd argument is optional = An URL preferably pointing to the WIKI. It's a future feature that is not currently working in
#									the current game version.

# COMMENTS
# Names of russian divisions are pretty long, so, choice was or make it like that or use shortenings (Gv. Pekh. Div. for example). Otherwise it would be just genericos (fallback names).
# Names are heavily based on soviet WW2 divisions with KRTL changes.
# Basically divisions has generic name, awards + "honoured name", which, as it's official name, used without brackets. Also, "honoured name" reflects history of the unit, not from which place division is (in main part of cases - sometimes it reflects from which place this division is).
# Exceptions here names "Siberian", "Far Eastern", "Alash", "Tukrestan", "Caucasian" etc. - which reflects exactly place of origin.
# Most of divisions are trying to follow for PDX pattern, which is naming divisions as they were named in the end of war (and such participated in battles which didn't happen on the start of game - so, PDX list can consist "Berlin Division"), but in this list mainly are used a border areas for naming, in which chances of fights will be very big. Otherwise there are used vague names as "ukrainian"/"belorussian" etc. which means almost anything.
# Overall pattern that Russian Army is using "Pekhotnaya" (Infantry) for Infantry Divisions, "Strelkovaya" (Rifle) is used only for special forces
# All infantry (+special forces) have the same namelist.
# Cavalry and cossacks also have overall same namelists, but cossack divisions have addition of the Host to names. Fallback name of Cossack divisions - "svodnaya" (combined) means that division is combined from different hosts.
# Mechanized (tank) and Tank divisions also have same namelist. Their names are based on Mechanized Corps or Brigades of USSR, since USSR reformed all their divisions after start of war.
# Guards namelists are divided on two - one is connected with infantry namelists, second with tank namelist. One list for all Guards division (infantry, motorized, mechanized) is here because it's just easier and we already have fuckton of namelists.
# Artillery namelists are WIP and can be removed in future (depends from how they will work - or will they work even). All Artillery divisions have same namelist, with difference in generic name.
# MVD divisions are NKVD divisions + militia and partisans.
# Pattern of awards: Red Banner order is replaced by Saint George Banner; orders of Suvorov, Kutuzov, Lenin etc. are replaced by orders of Saint Nicholas, Saint Vladimir and Saint Aleksandr Nevsky. Early divisions (first numbers) were mainly formed on basis of the RCW units, and such has awards for RCW - orders of Ice March, Liberation of Siberia, Liberation of Russia (fictional, inspired by Liberation of Siberia order), Volgian Attack (Fictional, refers to the Run to Volga), Moscow March (fictional, inspired by the Ice March order + KRTL history). Awards for RCW realistically will not appear in a later divisions, since these divisions didn't participate in RCW.
